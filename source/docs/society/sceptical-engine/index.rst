Sceptical Engine
=====================================

.. image:: /_static/images/room-sceptical-engine.png
   :alt: A small research workspace interior at night, viewed at eye level. A single researcher in shirtsleeves works at a long wooden workbench, focused on three monitors: the left shows an LLM chat interface with text bubbles, the right shows code analysis output and disassembly, the centre shows a hand-drawn flowchart pinned in place.

The Society's tooling arm. Built because responsible disclosure of sensitive firmware findings requires a
scanner that does not phone home, does not trust the model's confidence, and produces a verifiable chain of
custody from initial analysis through confirmed finding. The model proposes. The code decides. Nothing leaves
the host.
