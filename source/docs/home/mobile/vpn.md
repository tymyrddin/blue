# VPN on public Wi-Fi

Public networks carry traffic that anyone else on the same network can observe. A VPN encrypts the
connection between the device and the VPN server, preventing local eavesdropping.

## Reliable options

* [ProtonVPN](https://protonvpn.com/) (free tier available, audited, Switzerland-based)
* [Mullvad](https://mullvad.net/) (no account required, accepts cash payment, audited)
* [IVPN](https://www.ivpn.net/) (transparent, audited)

## How to use

* Install the app and create an account.
* Connect before joining any public network.
* On Android: Settings → Network & Internet → VPN → enable Always-On VPN for automatic protection.

Free VPN services that do not charge for the service tend to monetise through data collection. The
product is the traffic.
