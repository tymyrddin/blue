# When security tools get repurposed

Penetration testing frameworks like Cobalt Strike and Brute Ratel were built to help security teams simulate
adversarial behaviour: the red team runs realistic attack campaigns so that the blue team can test and improve
detection and response. These are legitimate and useful tools in that context.

Both have been cracked, leaked, and adopted by criminal operators. The same capabilities that make them useful
for red teamers, evasion of endpoint detection, flexible command-and-control communication, modular payload
delivery, make them attractive to attackers. The irony is not subtle: tools designed to improve security are now
used to compromise it, often against the very organisations that purchased them.

The practical consequence is that defenders cannot treat Cobalt Strike or Brute Ratel artifacts as reliable
indicators of sophisticated nation-state activity. Criminal groups use them routinely. Detection has had to
adapt accordingly.
