Lamplighter
=====================================

The Office's city-wide correlation effort. Not a project, officially. Several people call it one anyway, which
is how one knows it is real. Maps signalling paths, identifies choke points, detects coordinated influence
attempts that look like coincidence until they are lined up next to each other.
