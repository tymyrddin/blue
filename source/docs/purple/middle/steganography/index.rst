Steganography
=====================================

Steganography is the practice of hiding information so that its existence is undetectable.
For defenders, the problem is not decrypting a message but noticing that one exists at all.
This section covers the concepts behind covert channels, tools and techniques for detection
and extraction, and practice challenges.
