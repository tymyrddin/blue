Long Table
=====================================

The Establishment's assessment and review layer. Takes classified signals from the Quiet Room and
vulnerability findings routed from the Receiving Desk, correlates them across time and source, and
produces a single correlated picture: attributed infrastructure, campaign patterns, the vulnerabilities
in play, and analyst determination. The Long Table produces one view. It does not append alternatives.
